Mis Proyectos

QR code component solution

Solución del desafío QR Code Component Challenge de Frontend Mentor. Aquella plataforma es bastante útil para practicar posicionamiento y demás habilidades del desarrollo frontend. Realizado con HTML y CSS.

Seasons Countdown

Basicamente es una cuenta regresiva para cada una de las cuatro estaciones. La página indica días, horas, minutos y segundos faltantes para que ocurre el cambio de temporada. Esta realizado en HTML, CSS y JS.

To Read App

Es un proyecto que permite mantener un orden y registrar que libro/comic/manga se está leyendo en una base de datos. Funciona como un CRUD, y se interactúa desde la página con la base de datos. Esta realizado en JS, EJS, NodeJS y MySQL.
